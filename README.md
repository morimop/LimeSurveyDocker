LimeSurveyDocker
===
Overview

Docker scripts for running LimeSurvey with MySQL.

## Requirement
- docker-ce
- docker-compose
- git (optional)

## Quick Start
```
git clone https://github.com/morimop/LimeSurveyDocker.git
cd LimeSurveyDocker
./scripts/run_lime.sh
```
